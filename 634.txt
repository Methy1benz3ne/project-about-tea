木甑〔Wood steamer〕 做庄茶蒸茶工具。木制，高95 厘米，上口直径33厘米，下口直径43厘米，桶内近底部置 斗笠形蒸笼子，上加斗笠形竹制甑盖。作业时，每次向甑内 装入13〜15千克加工叶，盖上甑盖。用铁锅烧沸水发生蒸 汽，铁锅上置盖，盖上有通气孔。将木甑放在锅盖上，蒸到 甑盖上透大汽，盖缘滴水，甑内加工叶下陷时，表明蒸茶已 适度，即可从甑内倒出，进行遛茶（揉茶）。
遛板〔RoHiing board〕 做庄茶遛（揉）茶工具。一般 由6厘米厚的木板制成，长6米，宽1米，安装成30°的斜 坡，两边以竹竿做成扶手，以便遛茶。使用遛板遛茶是做庄 茶传统加工工艺，很原始，近年有茶厂以揉捻机代替遛板遛 茶。
六堡茶加工机具〔Liupu tea processing madiinery〕 六堡茶杀青、揉捻、渥堆和干燥等工序使用的各种机具。包 括杀青工序使用的锅式杀青机，揉捻工序使用的盘式揉捻 机，渥堆用的箩筐，干燥用的烘笼等。
乌龙茶加工机械〔OoSong tea processing machinery〕
乌龙茶加工机械的统称。通常包括做青机、杀青机、揉捻机、 烘干机、包揉机，用以分别完成晒青、摇青、杀青、揉捻、 初烘、初包揉、复烘、复包揉、足火等工序。其中，做青机 用于晾青和摇青或晒青；杀青机用于做青叶的杀青，通常使 用滚筒式杀青机;揉捻机用于杀青叶的揉捻;烘干机用于初 烘、复烘和足火工序，通常使用百叶链板式；包揉机用于初 烘和复烘后的初包揉、复包揉工序。做青机和包揉机是乌龙 茶加工机械中特有的机械。
红碎茶加工机械〔Black broken tea processing macWn-eir〕	红碎茶初制加工和精制加工所用机械的总称。初制
加工机械包括萎凋、揉切、发酵和烘干等工序所用的机械设 备，其中萎凋和发酵设备与工夫红茶初制设备通用。常用的 萎凋设备有萎凋槽和萎凋机等。发酵设备有车式和床式等。 烘干设备与工夫红茶、绿茶的初制、精制设备通用，常用者 有链板式和沸腾式烘干机等。揉切设备系红碎茶加工专用 机种，有盘式揉切机、转子揉切机、锤击机和齿辊揉切机等 形式。红碎茶精制加工机械与工夫红茶、绿茶精制机械基本 通用，只是工艺线路和筛网配置略有不同，包括筛分机械、 风力选别机械、拣梗机械和勻堆装箱机械等。
绿茶初制实验机械〔Experimental macfiine for green tea mini-manufacturing〕	用于绿茶工艺和茶树品种适制
性鉴定等模拟试验和少量茶样加工的实验设备。20世纪80 年代中期中国农业科学院茶叶研究所研制。由CS-30型锅 式杀青机、CR-16型揉捻机、CG-18型固样机、CC-30型锅 式炒干机、CCT-30型滚筒炒千机5台机器组成。全部机器 用单相交流电源驱动和提供热源，并使用温控仪和感温元
件组成的控制电路，对锅温或热风温度进行自动检测和控 制。可单台或成套使用。成套使用，按烘一炒一滚工艺用 CS-30型杀青机杀青，CR-16型揉捻机揉捻，CG-1S型固样 机烘二青，CC-30型锅式炒干机和CCT-30型滚筒炒干机 炒干，所制炒青绿茶品质良好。
红碎茶初制实验机械〔Experimental macWne for black broken tea mini-manufacturing]	用于红茶工艺和
茶树品种适制性鉴定等模拟试验和少量茶样加工的实验设 备。20世纪90年代初中国农业科学院茶叶研究所研制。由 小型转子揉切机、小型齿辊切茶机和微型盘式烘干机等机 器组成。小型转子揉切机，筒体内径8厘米；小型齿辊切茶 机，齿辊直径20. 5厘米，长20厘米，齿距3. 2毫米；微型 盘式烘干机的两层烘盘摊叶总面积为〇• 5米2,热风温度由 温控仪进行检测和控制。可单台或成套使用。成套使用作业 时，用转子揉切机对加工叶初切，用齿辊切茶机复切，用盘 式烘干机烘干，台时产量1〜2千克。
固样机〔SampSe fixing machine〕 茶叶鲜叶固样及少 量茶样烘干的实验设备。20世纪80年代中期中国农业科 学院茶叶研究所研制。由热源装置、烘叶器、温控装置、机 架和壳体组成。采用小型风机送入冷风，直接经电热丝加热 形成热风，通过烘叶器对茶叶进行固样烘干，热风温度由温 控电路进行检测和控制。在进行茶叶鲜叶样品固样时，投叶 量50〜100克。采用一次干燥法15分钟可以烘干，若采用 两段干燥法，中间摊凉20分钟，两段共烘10分钟，即可完 成一个样品的固样。
58型绿茶初制成套设备〔Complete set of green tea
processing equipment, Model 58D 绿茶初制加工成套设 备。1958年浙江省特产公司、浙江农学院、中国农业科学 院茶叶研究所联合研制。由双锅杀青机、双动式揉捻机、解 块分筛机、瓶式炒干机和双锅炒干机组成，铁木结构。I960 年正式定型，是中国最早定型的制茶成套设备和60年代广 大农村绿茶初制厂应用最广泛的设备。
58型红茶初制成套设备〔Complete set of black tea processing equipment, Model 58D	红茶初制加工成套设
备。1958年由湖北省研制。包括竹、木、铁结构的恩施58 型萎凋机、双动揉捻机、解块分筛机和万能干燥机。与此同 时，安徽、湖南和云南等省也相继试制出祁门干燥萎凋两用 机、安化烘茶箱、凤庆土烘房等简易烘干设备，在部分红茶 地区推广。均未正式定型。
67型绿茶初制成套设备〔Complete set of green tea processing equipment, Model 67D 绿茶初制加工成套机 械。20世纪60年代后期由杭州市农业局组织中国农业科 学院茶叶研究所、杭州茶叶试验场等单位，在58型绿茶初 制成套设备基础上联合研制。1967年定型。由双锅杀青机、 盘式揉捻机、解块分筛机、瓶式炒千机和双锅炒干机组成。
